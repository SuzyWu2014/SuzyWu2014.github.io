## Installation
+ fcitx
+ Sogou-pinyin

## Language Support

Go to `System Settings -> Language Support`

+ `Install/Remove Language`: add Chinese
+ `Keyboard input method system`: choose fcitx

Go to `System setting -> Text Entry`

+ add Sogou-pinyin
